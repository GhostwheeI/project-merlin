# Contributing to Ghostwheel

Thank you for your interest in helping improve Ghostwheel! Because the core application is closed-source, contributions focus on bug tracking, documentation, feature design feedback, and system presets.

## How to Contribute

1. **Submit Bug Reports**: If you find an issue in the pre-alpha builds, please search the active issues first, then file a report using the **Bug Report** template.
2. **Suggest Feature Enhancements**: We welcome feedback on agent behaviors, tool boundaries, and CLI parameters. Use the **Feature Request** template to propose designs.
3. **Share Configurations**: You can contribute custom model manifests, system prompts, or configuration presets for local hardware configurations in the community discussion board.

## Code of Conduct

We expect all contributors to maintain a respectful, collaborative, and professional environment across our community spaces.
